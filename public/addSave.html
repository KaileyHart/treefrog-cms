Create A New Navigation

To create a new navigation, type the name of your navigation in the input box below.
Then, click add data.



Update Content

To update content, simply click the "Get Data" button.
Then, type the change you want in the input box.
Next, click the navigation that you would like to change!




Delete Content

To delete content, simply click the "Get Data" button.
Then, type "delete" in the input box.
Next, click the navigation that you would like to delete.
Finally, refresh the page, click the "Get Data" button again, and view your changes!
